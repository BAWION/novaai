План доработки для реализации Use-case Дани (16 лет)
1. Онбординг и диагностика (2-3 дня)
 Базовый onboarding.ts - уже реализован
 Расширить диагностический квиз до 15 вопросов
 Создать полный набор вопросов с выявлением уровня Python, математики, этики ИИ
 Добавить визуализацию прогресса диагностики (шкала заполнения)
 Доработать алгоритм создания Skill DNA
 Реализовать детальный профиль с метриками по разным направлениям
 Добавить визуализацию Skill DNA в виде радар-диаграммы
2. Процесс обучения (3-4 дня)
 Базовая структура уроков - уже реализована
 Доработать формат micro-уроков
 Добавить поддержку видео-аватаров для объяснений
 Реализовать таймер 7-минутных уроков
 Расширить возможности чат-ассистента Nova
 Добавить педагогические сценарии для объяснения базовых концепций
 Интегрировать контекстную помощь на основе конкретных ошибок ученика
3. Практика в LabHub (4-5 дней)
 Создать базовую версию LabHub
 Интерфейс простого редактора кода
 Безопасная среда выполнения Python-кода
 Реализовать auto-grader для проверки упражнений
 Механизмы проверки базовых упражнений ("Hello World", переменные и др.)
 Разработать базу тестов для автоматической оценки
 Добавить геймификацию практики
 Мгновенные визуальные награды за правильные решения
 Цепочки заданий с нарастающей сложностью
4. Обратная связь (2-3 дня)
 Базовые подсказки через getProactiveHint - уже реализованы
 Реализовать Orb-ассистента
 Визуальный интерфейс с анимацией "шёпота" подсказок
 Интеграция с результатами квизов для контекстных подсказок
 Добавить систему адаптивной помощи
 Определение конкретных пробелов при неудачных квизах
 Генерация персонализированных подсказок через OpenAI API
5. Система достижений (3-4 дня)
 Отслеживание прогресса курса - уже реализовано
 Создать систему NFT-бейджей
 Разработать визуальный дизайн бейджей разных уровней
 Создать модуль для выдачи NFT-сертификатов
 Реализовать публичный профиль достижений
 Страница профиля с отображением всех достижений
 Экспорт достижений в социальные сети
6. Социальный компонент (2-3 дня)
 Добавить шеринг прогресса
 Генерация шаблонов для публикации достижений в VK и других сетях
 Интеграция кнопок "Поделиться прогрессом" в ключевые точки обучения
 Создать элементы геймификации и соревнования
 Локальные рейтинги между сверстниками
 Тематические "марафоны" для коллективного прогресса
Приоритеты реализации
Расширенная диагностика и Skill DNA (критично для правильного старта)
LabHub для практики кода (ключевой элемент для обучения программированию)
Orb-ассистент с интеллектуальными подсказками (поддержка при неудачах)
Система NFT-бейджей (мотивация для завершения треков)
Социальный компонент (вторичный приоритет)
Ресурсы
Фронтенд: React, Tailwind CSS, Framer Motion
Бэкенд: Node.js, Express, Postgres
AI: OpenAI API (GPT-4o для контекстных подсказок)
Визуальные элементы: Радар-диаграммы (D3.js), анимированные Orb-подсказки
Ожидаемый результат
Полноценная образовательная траектория для Даня, с диагностикой уровня, персонализированным обучением, практическими заданиями, интеллектуальными подсказками и системой мотивации через достижения и социальные взаимодействия.
