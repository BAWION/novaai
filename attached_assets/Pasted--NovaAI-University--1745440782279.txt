Какая «главная генеральная функция» NovaAI University

3. Как эта функция работает шаг-за-шагом

Шаг	Действие	Технологии
1. Onboarding	Вопросы → JSON профиля	React Form + Zod
2. Gap-анализ	Профиль × Skill Graph → список пробелов	Neo4j cypher / pgvector
3. Первая рекомендация	Rule («beginner» ⇒ Python-Zero) + ML-скор	FastAPI / Scikit-Learn
4. Обучение	Пользователь проходит урок, отправляет events	React Query mutation
5. Обратная связь	Tracker пишет время, ошибки, викторину	Prisma → progress
6. Адаптация	Re-score path каждые 3 урока или по fail ≥ 2	Background job (BullMQ)
7. Динамическое пополнение	CRON scraper → новый курс → новая планета	WebSocket push «new-cluster»

5. Что доработать в текущей схеме, чтобы ядро заработало

Пункт	Что добавить	Почему
Модель Skill	Skill { id, name, taxonomy } + связи Lesson↔Skill	Gap-анализ невозможен без нормализованных навыков
События	Унифицированный event learn.lesion.completed, learn.quiz.failed	ML-ранкер учится на единых данных
ML-флаги	Таблица feature_flags для A/B алгоритмов	Позволяет тестировать новую логику без отката кода
Explainability	В API ответе /recommend добавить reason	Прозрачность → доверие пользователя
