#### Рекомпозиция секций (десктоп ≥1280 px)

```
┌────────────────────────────── 100% width ────────────────────────────────┐
│  ← Breadcrumb (NovaAI / Панель)        🔍Cmd+K            🔔 уведомления │
└──────────────────────────────────────────────────────────────────────────┘
┌──────────── 70% ─────────────┬──────────── 30% ────────────┐
│  1. Орбитальная карта        │  Продолжить обучение (CTA)  │
│                              │  Предстоящий дедлайн        │
│                              │-----------------------------│
│                              │  Streak + XP-бар            │
└──────────────────────────────┼─────────────────────────────┘
┌──────────── 70% ─────────────┼──────────── 30% ────────────┐
│  AI-совет дня / Ошибки в коде│  Событие дня                │
│  (динамика)                  │  Форум-тема                 │
└──────────────────────────────┴─────────────────────────────┘
```

*На мобильном* — блоки идут **стеком**: CTA → streak → карта → событие → AI-совет.

### Приоритеты внедрения

1. **Сделать блок «AI-совет» динамическим**  
   - SQL-view «последний failed тест» → LLM prompt → короткая подсказка.  
2. **Карточка «Событие дня»**  
   - Берём ближайшее событие из таблицы `events` с датой ≥ today.  
3. **Hover-tooltips на орбитах**  
   - Tippy.js или Radix-Tooltip, показываем имя курса + прогресс.  
4. **Личная статистика**  
   - Cache-запрос `/stats/summary` — один легковесный API.  

Каждый пункт можно внедрять отдельно — если нужен точный prompt/код-дифф для Replit-агента, скажи какой именно блок берём первым.
