1. Sidebar (постоянная навигация слева)

Пункт	Назначение
Dashboard	Домашний экран: статистика прохождения, активные курсы
Roadmap	Персональный путь обучения (ветка модулей + прогресс)
LabHub	Интерактивные песочницы / ноутбуки, автопроверка кода
Courses	Библиотека всех курсов (фильтры: уровень, тег)
Community	Чаты, Discord‑виджет, AMA‑календарь
Profile	Данные пользователя, NFT‑сертификаты, streak‑счётчик
Settings	Язык интерфейса, тема, управление авторизацией
Logout	Выход и очистка токена
Адаптивность:

≥ 768 px — фиксированная колонка 64 → 80 px шириной, всегда видна.

< 768 px — скрыта за «гамбургером», выезжает поверх контента.

2. Content Area (правый сегмент, меняется роутером)

Роут	Основные блоки
/dashboard	• Orbital Widget — 3‑D карта актуальных курсов
• «Продолжить урок»
• Карточка streak + ближайшее достижение
/roadmap	• Древовидный граф модулей
• Прогресс‑бар каждого узла
• CTA «Начать следующий»
/labhub	• Таб «Задания» / «История»
• IDE‑окно, терминал, автотест‑лог
/courses	• Сетка карточек (+ фильтр)
• Детальная страница курса с описанием и видео‑тизером
/community	• Встроенный Discord‑iframe / Forum
• Таблица лидеров Daily Challenge
/profile	• Аватар, био, цели
• Таблица завершённых курсов + NFT‑ссылки
• Кнопка «Редактировать»
/settings	• Тема (Light/Dark), язык RU/EN
• Смена пароля, привязки OAuth
3. Header / Top Bar (опционально)
На десктопе может отсутствовать (минимализм).

На мобильном выводится ↗ кнопка «гамбургер», название текущего раздела, иконка уведомлений.

4. Навигационная логика
React Router v6 — ленивая загрузка каждого раздела (lazy(() => import(...))).

Zustand хранит глобальные данные пользователя (токен, прогресс) и доступен всем страницам.

Автоматический редирект: если нет JWT → /login; после login → /dashboard.

5. UX‑мелочи
Активный пункт ‑ мягкий неоновый градиент + лёгкое 3‑D поднятие (translateZ(4px) via Framer Motion).

Ховеры с «glass‑morphism» размытием.

На мобильных свайп от левого края открывает Sidebar‑drawer.

Эта архитектура обеспечивает единообразие навигации, лёгкое расширение новых разделов и дружелюбный опыт как на десктопе, так и в приложении‑PWA.
